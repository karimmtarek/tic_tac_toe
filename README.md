# TicTacToe

Human vs. AI Tic Tac Toe Game.

## Installation

- Clone this repo
- Change directory to repo root directory
- Run `$ bundle` to install missing dependencies
- Run `$ rake install` to install the gem

## Start the game

In your terminal type `tictactoe`
